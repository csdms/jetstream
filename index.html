CSDMS Demo Server

We've collected a set of Jupyter Notebooks that demonstrate how to use the software tools developed at CSDMS. The Notebooks are hosted on NSF's XSEDE Jetstream cloud computing service. To access the Notebooks, follow these steps:

  1. On your computer, open a terminal.
  2. In the terminal, use ssh to login to Jetstream with a csdmsXX username, where XX is in [01, 02, 03, ..., 25]: 

    3. $ ssh csdmsXX@js-104-142.jetstream-cloud.org

    You'll be prompted for a password, which you can get from a CSDMS IF staff member (or email csdmssupport@colorado.edu if a staff member isn't present).

  3. Once logged in to Jetstream, a Jupyter Notebook server automatically starts. When it completes, it will print a (long) URL. Copy/paste the URL into a web browser on your computer to start a Jupyter Notebook session.
  4. On the Jupyter Notebook start page, click on the README.ipynb file. It gives a listing and brief description of all Notebooks available on the server. Click on a link to access a Notebook.
  5. When finished, type Ctrl-C twice in the terminal to close the Jupyter Notebook server, then type exit to exit Jetstream.

Any questions? Please email csdmssupport@colorado.edu.
